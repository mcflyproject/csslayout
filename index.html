CSS Layout Cheatsheet

Box Models

Standard Box Model
Part Name Definition Sizing
Content Box The area where your content is displayed. Size it using properties like inline-size and block-size or width and height
Padding Box The area around the content as white space. Size it using padding and related properties.
Border Box Wraps the content and any padding. Size it using border and related properties
Margin Box The outermost layer, wrapping content, padding, and
border, as whitespace between this box and other elements.		 Size it using margin and related properties.

Layout Properties

Layout Properties
Property Definition Types or Values
Display Change how elements behave in normal flow. block, inline, or inline-block
Floats Causes block level elements to wrap along one side of an element. right, left
Clear Used with float property to keep one side of an element clear of other elements.
Usually used after the element that was floated.		 left, right, both
Position Precise control of position of boxes inside of boxes. e.g. static, absolute, fixed, sticky
